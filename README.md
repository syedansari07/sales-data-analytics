📊 Sales Dashboard (Tableau)
📌 Project Overview

This project presents an interactive Sales Dashboard built using Tableau to analyze sales performance over time.
The dashboard provides insights into sales, profit, quantity, category performance, and weekly trends to support data-driven business decisions.

The dashboard allows users to quickly monitor key performance indicators (KPIs) and compare current year performance with previous year metrics.

🎯 Project Objectives

  Analyze overall sales performance
  
  Compare current year vs previous year
  
  Identify top performing product categories
  
  Track weekly sales and profit trends
  
  Provide visual insights for decision making

  📊 Key Metrics in the Dashboard

The dashboard highlights the following KPIs:

  Total Sales: $733K
  
  Total Quantity: 12K units
  
  Total Profit: $93K

Each KPI includes a percentage comparison with the previous year.

💡 Key Insights

  Sales increased compared to the previous year.
  
  Certain categories like Phones and Chairs contribute significantly to revenue.
  
  Weekly sales patterns show fluctuations across months.
  
  Profit margins vary across product categories.
